# RumahKita AntiXray

Ore security protection system against *X-Ray hackers*. Hides valuable ores from client-side hacks using specific obfuscation/replacement methods.

## Core Features
*   **Packet Handling:** Manipulates visual blocks before they are sent to the player to prevent X-Ray rendering.
*   **Alerts:** Notifies Admins if a player mines an unreasonable (suspicious) amount of ores.

## Dependencies
*   `ProtocolLib`
