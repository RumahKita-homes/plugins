# RumahKita DiscordVerify

Minecraft account synchronization system with roles on the RumahKita Discord server.

## Core Features
*   **In-Game Link:** Players can link their Discord accounts to receive rewards or specific roles.
*   **Bot Integration:** Interacts directly with the RumahKita Discord Bot.
*   **Role Sync:** Automatic synchronization of donor or in-game ranks to the Discord server.

## Commands
*   `/discord` or `/link` - Link a Discord account.
*   `/unlink` - Unlink the account.
