# RumahKita FarmAI

Virtual farming assistant. Helps automate certain aspects of agriculture or land clearing (such as replanting or harvesting).

## Core Features
*   **Auto Replant:** Automatically replants crops immediately after they are broken.
*   **AI Assistants:** Virtual entities/robots that assist with farming activities (based on level/rank).

## Commands
*   `/rkfarmai` - Main FarmAI menu.
