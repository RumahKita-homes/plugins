# RumahKita Guilds

Custom faction or group system for RumahKita Server. Combines social, economic, and competitive features among players.

## Core Features
*   **Guild Creation:** Players can create their own Guild using their Economy balance.
*   **Guild Roles:** Includes *Leader*, *Co-Leader*, and *Member* ranks.
*   **Leveling System:** Guilds can be leveled up to unlock additional member slots.
*   **Guild Bank:** A communal bank for guild member contributions.
*   **Guild Chat:** Private chat channel exclusively for guild members.

## Commands
*   `/guild` or `/g` - Access the main Guild menu/commands.
*   `/g create <name>` - Create a new Guild.
*   `/g invite <player>` - Invite a player to the Guild.
*   `/g chat` - Toggle guild chat mode.

## Dependencies
*   `RumahKitaEconomyV2` (Or Vault)
