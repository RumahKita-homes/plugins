# RumahKitaMinigames

This is the consolidated Minigames plugin for the RumahKita Minecraft server. It combines multiple gameplay features into a single efficient module:

- **Games** (Coinflip, Rock Paper Scissors)
- **PvP** (1v1 Arena, Duels)
- **Capture The Flag** (CTF events)
- **Fishing** (Custom fishing mechanics and rewards)

## Features
- **Coinflip & RPS**: Wager economy money against other players.
- **PvP Arena**: Integrated 1v1 duel system with kit selection.
- **Capture The Flag**: Event-based team pvp minigame.
- **Custom Fishing**: Rarity-based fishing system with economy rewards.

## Requirements
- Paper 1.20.4+
- PlaceholderAPI (Optional/SoftDepend for CTF and Fishing placeholders)
- Vault (Optional/SoftDepend for Economy integration)
- RumahKitaEconomyV2 (Recommended for seamless currency handling)
