# RumahKita PvP 1v1

Duel/Matchmaking system for one-on-one player combat in custom arenas.

## Core Features
*   **Arena System:** Matches take place in isolated arenas to prevent outside interference.
*   **Kit Selection:** Pre-defined equipment sets for duels (e.g., Diamond Armor, Iron Armor, etc.).
*   **Betting System:** Players can wager money on the outcome of a duel.
*   **Elo/Rating:** Players maintain a competitive win/loss rating.

## Commands
*   `/duel <player>` - Challenge another player to a duel.
*   `/1v1` - Queue up to find a random opponent.
