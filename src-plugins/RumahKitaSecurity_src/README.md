# RumahKitaSecurity

This is the core security and moderation plugin for the RumahKita Minecraft server. It combines several previously separate plugins into one hardened security suite.

- **AntiCheat**: Detects and prevents unauthorized movement and combat modifications.
- **AntiXray**: Advanced block obfuscation and mining alerts.
- **SecurityBan**: Strict moderation tool for IP/CIDR bans and UUID blacklists.
- **RamGuard**: Automated memory cleanup and optimization utility.
- **OreSpectator**: Allows staff to monitor player mining activity dynamically.

## Features
- Comprehensive player movement and combat checks.
- Real-time logging of suspicious activity.
- Subnet and CIDR level IP ban system.
- Memory leak prevention and RAM optimization tools.
- Specialized spectator modes for catching X-Ray users.

## Requirements
- Paper 1.20.4+
