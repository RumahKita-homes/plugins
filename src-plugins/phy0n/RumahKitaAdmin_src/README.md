# RumahKita Admin

Core administration and moderation system for RumahKita Server. Manages chat functions, Anti-VPN system, scheduled restarts, punishments (Mute, Jail, Freeze, Kick, Ban), etc.

## Core Features
*   **Account Security:** Prevents alt limit bypasses and blocks VPN/Proxy connections.
*   **Chat Manager:** Anti-Capslock, Anti-Spam, and Profanity Censor.
*   **Moderation Utilities:** Vanish, Freeze, Invsee, God Mode, and Spy Command.
*   **Auto Restart:** Automates scheduled restarts with countdown broadcasts.

## Commands
*   `/rka` - Main admin command (mute, freeze, invsee, checkalts, vpn, etc)
*   `/sc` - Staff Chat (Exclusive staff communication)

## Permissions
*   `rumahkita.admin` - Access to all moderation features.
