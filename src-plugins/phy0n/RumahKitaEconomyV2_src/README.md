# RumahKita Economy V2

Custom currency (Rupiah) plugin built specifically for RumahKita. Replaces conventional economy plugins and integrates a dynamic *Global Market* system, daily sell limits, and other features.

## Core Features
*   **Global Market:** GUI-based shop driven by stock and demand (prices fluctuate dynamically).
*   **Wealth Distribution:** `/payall` feature to distribute money to all online players.
*   **Leaderboards:** `/baltop` system with options to hide balance (`/hidebal`).
*   **PAPI & Vault:** Fully integrated with PlaceholderAPI and Vault.

## Commands
*   `/market` or `/shop` - Opens the Global Market GUI.
*   `/pay`, `/payall` - Money transfer and distribution.
*   `/bal`, `/baltop`, `/hidebal` - Balance utilities.
*   `/sellhand`, `/sellall` - Quick selling mechanics.
*   `/rke` - Admin commands (set, give, take, voucher).

## Dependencies
*   `Vault`
*   `PlaceholderAPI`
