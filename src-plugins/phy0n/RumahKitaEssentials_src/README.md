# RumahKitaEssentials

This is the consolidated Essentials module for RumahKita, combining various previously separate features like Admin commands, Warps, Utilities, Carry, Bansos, Sleep, and Vanish.

## Features
- **Admin Utilities**: Staff chat, essential server management tools.
- **Warps System**: Server warps, player warps, and randomized teleportation (RTP).
- **Quality of Life**:
  - Carry mechanism for players and entities.
  - Sleep skipping logic.
  - Custom Vanish (particle-free) functionality.
  - Friday "Bansos" distribution.

## Dependencies
- PlaceholderAPI
- Vault
- RumahKitaEconomyV2 (Soft)

## Build Instructions
1. Navigate to this directory.
2. Run `mvn clean package`.
3. The resulting `RumahKitaEssentials-1.0.0.jar` will be available in the `target/` directory.
