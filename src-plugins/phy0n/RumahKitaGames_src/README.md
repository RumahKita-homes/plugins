# RumahKita Games

Interactive mini-games and betting plugin between players. Utilizes a secure GUI system to allow players to safely gamble their balances.

## Core Features
*   **Coinflip:** Toss a coin with a true 50/50 chance against other players.
*   **Rock Paper Scissors:** Play the classic game against others.
*   **Server Tax:** Implements a tax deduction on winnings to prevent economy inflation.

## Commands
*   `/cf` - Opens the Coinflip menu.
*   `/rps` - Opens the Rock Paper Scissors menu.

## Dependencies
*   `RumahKitaEconomyV2`
