# RumahKita Trade

A secure trading system between players via GUI. Prevents scams and eliminates the need to drop items on the ground.

## Core Features
*   **Anti-Scam GUI:** Players can review items securely before pressing the "Accept" button.
*   **Currency Exchange:** Players can add Rupiah directly into the trade session.
*   **Strict Security:** *Shift-click* and *Drag-click* events are completely disabled to prevent item duplication.

## Commands
*   `/trade <player>` - Send a trade request to another player.
*   `/trade accept` - Accept an incoming trade request.
*   `/trade deny` - Deny an incoming trade request.

## Dependencies
*   `RumahKitaEconomyV2`
