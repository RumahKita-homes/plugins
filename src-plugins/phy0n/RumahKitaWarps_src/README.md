# RumahKita Warps

Custom teleportation system encompassing Server Warps, Player-created Warps (PWarp), Random Teleport (RTP), and Back commands.

## Core Features
*   **Server Warps:** Admins can create public locations using a categorized GUI.
*   **Player Warps (PWarp):** Players can rent and publish their own locations with a per-click tariff.
*   **Random Teleport (RTP):** Safe random teleportation feature to explore new lands.
*   **Back System:** Return to the previous location after dying or teleporting.

## Commands
*   `/warp`, `/setwarp`, `/delwarp`, `/editwarp` - Server Warp management.
*   `/pwarp` - Opens the Player Warps GUI.
*   `/rtp` - Random teleportation to the wild.
*   `/back` - Returns to the last location (pre-death/pre-teleport).

## Permissions
*   `rumahkitawarps.admin` - Server warp management.
